## Project Overview

This project demonstrates how Prompt Engineering can be used to generate professional website copy for a local café. Using ChatGPT, structured prompts were designed to create engaging, conversion-focused, and customer-friendly content.

The generated website copy includes:

- Homepage
- About Us
- Services
- Call-to-Action (CTA)
- Prompt Framework
- Prompt Outputs

The objective is to create website-ready marketing content while documenting the prompt engineering workflow.
