# Prompt Iterations

## Iteration 1

### Prompt

Write a homepage for a local café.

### Result

The content was simple but lacked branding and persuasive marketing.

---

## Iteration 2

### Prompt

Write a professional homepage for Brew Haven Café targeting students, families, and professionals.

### Improvements

• Better tone

• Stronger headline

• More engaging introduction

• Customer-focused writing

---

## Iteration 3

### Prompt

Act as an expert website copywriter.

Generate complete homepage content with:

- Hero section
- About
- Benefits
- Services
- Testimonials
- CTA

The tone should be friendly, professional, and SEO-friendly.

### Result

Generated high-quality website copy suitable for a modern café website.

---

## Learning

Prompt refinement significantly improved the quality, structure, and usefulness of the AI-generated content.

Adding audience, tone, business goals, and formatting instructions produced more professional outputs.
